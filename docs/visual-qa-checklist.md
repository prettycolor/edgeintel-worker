# Visual QA Checklist

Use this before any final visual polish pass.

## Provider Workspace

- Preset selection changes the capability panel immediately.
- Auth strategy guidance matches the selected provider.
- Secret health copy never implies unsupported OAuth.
- Empty, create, selected, and error states all read cleanly.

## Tunnel Workspace

- Zone discovery, hostname validation, and provisioning states are visually distinct.
- Test history and failure deltas are readable without raw JSON dependence.
- Pairing and connector states are obvious at a glance.

## Desktop Connector

- Pairing, install, runtime, degraded, and error states are all visually separate.
- The next required action is always obvious.

## Commercial Output

- Cloudflare fit, access hardening, latency, and resilience scores are visible first.
- Customer-safe and operator narratives do not blur together.
- Expansion candidates read like a ranked SE motion list.
